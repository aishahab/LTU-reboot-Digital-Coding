The following information was taken from w3schools, I learn a lot from this website:
The <head> element contains meta information about the HTML page
The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
The <h1> element defines a large heading
The <p> element defines a paragraph

27/10/20 - HTML & CSS

Today, I learnt how CSS works from top to bottom, so any code that is at the bottom, overrides anything that may be specified further up.
I learnt that:

  1. #id selector styles the element with the specified id
  2. The div seperate each section of code from another section of code - can use this for nesting
  3. @fontface = imports a font
  4. When selecting fonts in our code, we can list fonts. Therefore, if one font can't be loaded in a specific browser, then it will try to load one of the other fonts
